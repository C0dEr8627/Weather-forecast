# Weather Forecast App

A responsive web application that provides real-time weather information and a 5-day weather forecast for any city worldwide. The app dynamically displays weather data including temperature, humidity, UV index, air pressure, wind speed, and Air Quality Index (AQI), with a clean and visually appealing UI.

---

## Features

- **Search** weather by city name.
- **Current weather details:** temperature, humidity, UV index, air pressure, wind speed, and AQI.
- **5-day forecast** with temperature and weather conditions.
- **Dynamic weather icons** to match current conditions.
- **Responsive design** optimized for desktop and mobile.
- Smooth gradient background with card-based UI.
- Easily customizable color themes using CSS variables.
